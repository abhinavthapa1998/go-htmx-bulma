# Go + HTMX + Bulma CSS

A little project to learn templating with Go using Bulma CSS

## API Reference

#### Get all items

```http
  POST /add-films
```

| Parameter  | Type     | Description                        |
| :--------- | :------- | :--------------------------------- |
| `title`    | `string` | **Required**. title of the film    |
| `director` | `string` | **Required**. director of the film |
